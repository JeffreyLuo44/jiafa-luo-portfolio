# Jiafa Luo Portfolio (React + Vite)

This project is a React app built with Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

If deploying as a GitHub Pages project site, set `base` in `vite.config.js` to your repository path, for example `/your-repo-name/`.
